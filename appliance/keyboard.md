# 键盘对比

机械键盘似乎有个规律，就是越小，手感越好。

## 有线机械键盘

- Filco忍着104：手感流畅，外形好看，ABS键帽打油严重，性价比低，但手感和做工基本挑不出毛病；
- 雷柏V500S Pro：性价比首选；
- 鸭子2108S：RGB108款用久后手感好，灯比较不错。
- 凯酷87：外形有机械键盘感觉，但手感比较平庸；
- 凯酷104：无激情的流畅手感；
- PokeII：手感极佳。但是没有方向键；
- HHKB：手感比较肉，没有性价比。

## 蓝牙键盘

- 罗技K380：iPad、手机首选，手感好于几十元价位
- Filco蓝牙：手感极佳
- 雷柏79元无线键鼠套装：键盘在少量输入场景下，手感好。键帽较硬，大量输入错误率高，废手。
